# Health Assistant

Health Assistant is a local fitness and diet tracking MVP for the 2026-07-06 to 2026-08-26 fat-loss training cycle.

The current implementation includes:

- React + Vite frontend.
- Python local API server.
- SQLite local database.
- Weekly training plan.
- Daily check-ins.
- Custom tasks.
- Meal logging and calorie estimation.
- Optional LLM meal and coach advice through `.env`.
- Workout logging.
- Statistics dashboard.

## Project Structure

| Path | Purpose |
| --- | --- |
| app/ | Runnable application |
| app/client/ | React + Vite frontend source |
| app/client/dist/ | Frontend build output, generated by `npm.cmd run build` |
| app/server.py | Local Python API and static file server |
| app/data/fitness-assistant.sqlite | Local SQLite database |
| docs/ | Requirements, roadmap, architecture, plugin recommendations |

## Quick Start

```powershell
cd D:\DevEnv\Work\health-assistant\app
npm.cmd install --cache .\.npm-cache
npm.cmd run build
python server.py
```

Optional AI config:

```powershell
cd D:\DevEnv\Work\health-assistant\app
copy .env.example .env
notepad .env
```

Fill `LLM_API_KEY`, `LLM_BASE_URL`, and `LLM_MODEL` for any OpenAI-compatible Chat Completions provider. `.env` is ignored by Git.

Open:

```text
http://127.0.0.1:8765
```
